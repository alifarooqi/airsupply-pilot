# airsupply-pilot

## Description

A Django app build for Hospital Authority (Hong Kong) as part of our Software Engineering course project.

## Installation

1. Download dependencies:
`pip install`

2. Migrate database
`python manage.py migrate`

3. Run:
`python manage.py run`
